## 🚀 Getting Started

### 📋 Prerequisites
- Node.js (v18+) `>=18.x`
- npm (v9+) `>=9.x`
- Git
- DeepSeek R1 API key

### ⚡ Installation

## ⚙️ Running the Swarm
```bash
npm run dev
```

## 💬 Interacting with Agents
1. Start the development server:
```bash
npm run dev
```
2. Visit `http://localhost:3000/` in your browser
3. Use the chat interface:
```bash
/ai "Your message here"
```
